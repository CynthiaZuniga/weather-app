Weather App
Proyecto realizado para el módulo Frontend del bootcamp.
La aplicación muestra información básica del clima de distintas ciudades y permite ver el detalle de
cada una.
Funcionalidad
- Lista de ciudades con temperatura y estado del clima
- Vista de detalle por ciudad
- Pronóstico semanal
- Navegación entre páginas usando localStorage
Tecnologías usadas
- HTML
- CSS
- SASS
- JavaScript
- Bootstrap
Estructura del proyecto
weather-app/
index.html
detail.html
js/app.js
scss/
Conceptos aplicados
- Manipulación básica del DOM
- Uso de arreglos y ciclos
- Metodología BEM
- Organización de estilos con SASS
Uso del proyecto
Abrir index.html y seleccionar una ciudad para ver el detalle del clima
